MARV 非公式ドキュメントへようこそ！

## このドキュメントの目的

MARV の公式ドキュメントは、様々な事情により情報が欠けていたり、古くなっていたりします。

その結果、新参プレイヤーはもちろん、ずっと前から遊び続けているユーザーでさえも MARV の仕様を理解していない場合が多いです。

このドキュメントは、欠けている情報を補うために作られました。

## あなたの貢献が必要です！

このドキュメントは[GitHub](https://github.com/ruka-64/marv-unofficial-docs/)で公開されています。

私一人で全ての情報を収集し、まとめることは難しいです。あなたの Pull Request をお待ちしております！
